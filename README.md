## 使用

需安装 Python 包管理器：[Poetry](https://python-poetry.org/docs/#installation)

```powershell
git clone --depth=1 "https://github.com/lengyanyu258/ArknightsWordCount.git"

cd ArknightsWordCount

poetry shell

poetry install

python main.py -h
```
